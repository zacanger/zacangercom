KineticJS: this is unmaintained now, but it's a pretty swell library for working with canvas interactively. It looks like a lot of those kind of interactive things use that (InStyle, for example, or the place where I buy eyeglasses).

The source of this embeddable widget would be worth examining, for sure. It's a LOT, but stripping away all the extras there are probably about 10 js file really worth digging into.

As far as Angular-specific drag and drop, Dragula is by far the best (and easiest) I've tried to play with.

The 6px/Cloud6 service has very powerful image processing... but maybe overkill?

Literally Canvas is almost literally canvas, plus some awesomeness. They have demos involving images as the background, and make it easy to work with uploads/downloads and with Imgur.

Imgix might be closer to that scale.

Uploads:
There are some basics of uploads here and here, but it basically comes down to:
<FORM METHOD=POST ENCTYPE="multipart/form-data" ACTION="../cgi-bin/mycgi.pl"> File to upload: <INPUT TYPE=FILE NAME="upfile"><BR> <INPUT TYPE=SUBMIT VALUE="Submit"> </FORM>
with some cgi, or without:
<form method="post" action="file-upload-1.htm" name="submit" enctype="multipart/form-data"> <input type="file" name="fileField"><br><br> <input type="submit" name="submit" value="Submit"> </form>
And then figuring out things with Firebase or Parse or whatever for storage.
